***QUICK START***

1. Right-click on any folder with music or a playlist;
2. Choose "Open in Meson Player" from a context menu.


***HOTKEYS***

NumPad 5 - stop/play
NumPad 0 - pause/resume
NumPad 4 - previous track
NumPad 6 - next track
NumPad 4 (long press) - rewind
NumPad 6 (long press) - fast forward
NumPad 7 - jump to previous folder
NumPad 9 - jump to next folder
NumPad 8 - volume up
NumPad 2 - volume down
NumPad 1 - system volume down
NumPad 3 - system volume up

For more info, open the application's tray context menu and select Help > Manual.
